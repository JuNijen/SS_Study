MongoDB는 C++로 작성된 오픈소스 문서지향(Document-Oriented)적인 Cross-platform 데이터베이스이다.

## **[ 특징 ]**

- NoSQL (Not Only SQL)
- Document Oriented
    
    동적인 스키마를 가지고 있다. 서로 다른 데이터들을 가지고 있을 수 있다.
    
- Collection
- Database
- RDMS(Relational Database Management System)와의 비교
